# Forward/Deferred Rendering Engine



Requirements
------
* Visual Studio 2022 or newer
* CMake VERSION 3.13

Setup
------ 
* ``` git clone https://github.com/juice-artur/OpenGL_Research.git --recursive```
* ```git submodule init .```
* ```cd OpenGL_Research```
* ```cmake -B build -G "Visual Studio 17 2022"```
* ```cmake --build build```

Movement
-----
* WSDA and mouse controll
